# What is it?

It is a motion senor garbage can. If the sensor detects anyone standing in front of the garbage can,
it opens. Very simple.

# Parts

- Arduino UNO
- Ultra-Sonic sensor
- Servo motor
- Lego Technic pieces
- A garbage can ...
